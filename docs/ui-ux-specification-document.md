# AI图像生成与超分工具 UI/UX 规格说明书

本文档为跨平台 Tauri 桌面应用提供一套完整设计规范、交互标准与响应式布局规则。本应用针对中、大屏做动态适配（设置窗口最小尺寸强约束），同时支持系统原生浅色 / 深色双主题。

---

## 1. 窗口与响应式设计规范

为容纳复杂参数控制面板与高密度图像生成画布，本应用仅针对桌面端屏幕做专属优化。

### 1.1 窗口尺寸限制

- **最小允许尺寸**：`1024 × 720 像素`（窗口管理器强制该下限，避免界面元素裁切）
    
- **默认启动尺寸**：`1280 × 800 像素`（适配主流笔记本基础分辨率）
    
- **适配目标视口**：中等屏（宽度 1024px–1440px）、大屏（宽度 1440px 及以上）
    

### 1.2 布局栅格与自适应逻辑

界面采用三栏式 CSS 网格 / 弹性盒布局，可流畅缩放，同时保证控制面板内容清晰可读。

```Plain
+-----------------------------------------------------------------------+
| 窗口自定义标题栏（集成导航栏 / 窗口拖拽区域）                           |
+---+----------------------------+--------------------------------------+
| 全 | 控制面板                  | 工作区 / 图像画布                     |
| 局 | （固定宽度区间：360px–400px） | 自适应区域：填充剩余全部可视空间      |
| 导 |                            |                                      |
| 航 | - API/服务商选择下拉        | - 高清图像预览窗口                    |
| 栏 | - 提示词输入（正向+反向负面） | - 对比滑动条（原图/超分效果图切换）   |
| 竖 | - 生成参数调节滑块          | - 功能操作栏（下载、图像超分）        |
| 边 |                            | - 生成任务队列 / 历史记录列表         |
| 栏 |                            |                                      |
|    | 【底部主题切换按钮】        |                                      |
+---+----------------------------+--------------------------------------+
```

1. **全局导航侧边栏（最左侧）**：窄幅常驻侧边栏（宽度 64px），用于切换功能模块（图像生成、API 配置、系统设置），底部放置主题切换按钮。
    
2. **控制面板（中左区域）**：半固定宽度（360px–400px），承载输入框、参数滑块、下拉菜单与生成操作按钮。固定宽度可避免超宽屏幕下输入组件拉伸变形。
    
3. **工作区 / 画布（右侧区域）**：完全自适应弹性布局，占用所有剩余横竖可视空间，最大化展示生成图与超分效果图。
    
4. **屏幕适配规则**
    
    1. 中等屏幕（1024px–1440px）：底部生成队列以单行横向滚动轮播展示；画布仅展示单张选中主图。
        
    2. 大屏（1440px 及以上）：画布支持分栏并排展示原图与超分对比图，或加载 2×2 多图生成网格，无需隐藏左侧参数面板。

## 2. 主题与视觉规范

系统配套两套高对比度主题，统一所有组件层级视觉逻辑。

### 2.1 色值规范表

|   |   |   |   |
|---|---|---|---|
|设计标记 Token|深色主题（默认）|浅色主题|适用场景|
|应用主背景|`#0E0F12`|`#F8FAFC`|窗口底层底色|
|面板表层底色|`#181A20`|`#FFFFFF`|控制面板、输入区、卡片容器|
|面板悬浮底色|`#22252D`|`#F1F5F9`|悬浮卡片、列表项 hover 背景|
|边框 / 分割线|`#2E333F`|`#E2E8F0`|输入框边框、面板分隔条|
|一级正文文字|`#F3F4F6`|`#0F172A`|核心可读文本、模块标题|
|二级辅助文字|`#9CA3AF`|`#64748B`|标签、小字说明、提示文案|
|主强调色|`#6366F1`（靛蓝 500）|`#4F46E5`（靛蓝 600）|交互按钮、激活高亮状态|
|成功状态色|`#10B981`（翡翠绿 500）|`#059669`（翡翠绿 600）|校验通过、操作成功提示|
|警告 / 错误色|`#EF4444`（红色 500）|`#DC2626`（红色 600）|参数报错、接口连接失败|

### 2.2 字体层级规范

统一字体栈，调用系统原生渲染引擎，保证桌面端原生质感。

- **字体族配置**
    
    - macOS：苹方体系 `-apple-system`、`BlinkMacSystemFont`
        
    - Windows：`Segoe UI`
        
    - Linux：`Inter`、通用系统字体 `system-ui`
        
- **字号层级**
    
    - 模块标题：16px，字重 600（半粗体）
        
    - 组件标签：13px，字重 500（中等字重）；设置类标签字母加宽字距
        
    - 正文 / 输入框文字：13px，字重 400（常规）
        
    - 注释 / 悬浮提示：11px，字重 400（常规）
        

### 2.3 主题切换交互规则

1. **切换入口**：放置在全局侧边栏下半区域，使用太阳 / 月亮双态图标区分明暗。
    
2. **切换逻辑**
    
    1. 单击图标直接切换浅色 / 深色模式；
        
    2. 可选扩展下拉菜单：增加「跟随系统主题」选项。
        
3. **过渡动画**：全局组件统一添加平滑 CSS 过渡：`background-color 0.2s ease`、`border-color 0.2s ease`，切换时无生硬闪屏。
    
4. **偏好持久化**：用户所选主题本地持久存储（Tauri 存储引擎 / 本地缓存），应用启动渲染窗口前先行加载，避免初始无样式闪烁。

## 3. 自定义标题栏 & 跨操作系统适配规范

为实现全系统统一界面，采用无边框窗口 + 一体化自定义顶部标题栏方案。

```Plain
+-----------------------------------------------------------------------+
| [macOS原生红绿灯控制区]             窗口拖拽区域              [_] [□] [X] |
+-----------------------------------------------------------------------+
```

1. **macOS 原生窗口控件适配**
    
    1. 左上角预留 80px 宽、40px 高空白区域，承载系统原生关闭 / 最小化 / 全屏圆点控件；
        
    2. 该区域附近不放置可交互按钮，防止操作冲突。
        
2. **Windows / Linux 自定义窗口控件**
    
    1. 右上角渲染配套应用风格自定义按钮：最小化`_`、最大化`□`、关闭`X`；
        
    2. 悬浮状态区分高亮：关闭按钮 hover 时填充红色`#EF4444`底色。
        
3. **窗口拖拽能力**
    
    1. 标题栏空白背景区域添加 `data-tauri-drag-region` 属性，支持拖动窗口；
        
    2. 标题栏内可交互控件（搜索框、设置标签）排除拖拽属性，保证点击功能正常。

## 4. 交互规范与用户操作流程

### 4.1 流程 A：API 服务商配置

用户填写并校验外部图像生成、超分接口密钥。

```Plain
+-----------------------------------------------------------------------+
| 【API设置面板】                                                        |
|                                                                       |
| 服务商选择下拉框：[ Stable Diffusion Web API  ▼ ]                      |
|                                                                       |
| 接口地址URL输入框：[ https://api.provider.com  ]                       |
|                                                                       |
| API密钥输入框：[ •••••••••••••••••••• 【眼睛显示/隐藏图标】 ]          |
|                                                                       |
|                           【测试连接】按钮                              |
+-----------------------------------------------------------------------+
```

1. **密钥脱敏输入**：密钥默认密码掩码隐藏；点击眼睛图标明文展示密钥。
    
2. **连接校验流程**
    
    1. 点击「测试连接」后按钮进入加载态，显示环形加载动画；
        
    2. 校验成功：按钮底色切换为绿色并展示对勾图标，弹出短时通知提示连接正常，同时展示远端可用模型列表；
        
    3. 校验失败：输入框边框标红，下方输出诊断错误文案（例：`权限校验失败(401)：请核对API密钥`）。
        
3. **自动后台保存**：用户停止输入 500ms 防抖延迟后，自动写入本地加密配置文件；界面角落小字提示保存完成。
    

### 4.2 流程 B：提示词填写与图像生成布局

控制面板简化提示词编写与参数设置流程。

1. **多行提示词输入区**
    
    1. 主文本框填写正向生成提示词；下方折叠面板存放反向负面提示词；
        
    2. 快捷键 `Cmd/Ctrl + Enter` 一键启动图像生成。
        
2. **生成全生命周期交互**
    
    1. 接口请求期间，「生成」主按钮置灰禁用，按钮内实时展示已耗时；
        
    2. 画布区域展示骨架加载动画，支持进度百分比展示；
        
    3. 常驻「取消任务」按钮，可终止卡死任务，不冻结应用主界面。
        

### 4.3 流程 C：图像超分（超分辨率放大）

画布加载原图后，用户可一键启动高清放大处理。

```Plain
+-----------------------------------------------------------------------+
| 【画布主预览区】                                                        |
|                                                                       |
|  +-----------------------------------------------------------------+  |
|  |                             |                                   |  |
|  |                             |                                   |  |
|  |       原图预览              |<-- 拖拽分割线 --> 超分效果图       |  |
|  |                             |                                   |  |
|  |                             |                                   |  |
|  +-----------------------------------------------------------------+  |
|                                                                       |
|  [超分模型选择 ▼]  [放大倍数：2倍 | 4倍] 【执行超分辨率放大】按钮       |
+-----------------------------------------------------------------------+
```

1. **快捷操作入口**：鼠标悬浮历史队列任意生成图，弹出悬浮快捷栏，内置「超分」按钮；点击后原图加载至主画布。
    
2. **超分配置工具栏**：预览图下方常驻操作栏，可切换超分模型、选择放大倍率（2 倍 / 4 倍）。
    
3. **分栏对比滑动器（视觉校验功能）**
    
    1. 超分任务完成后，预览区自动切换为左右分栏滑动对比布局；
        
    2. 中间竖线为拖拽分割器：向左拖动显示原图细节，向右拖动展示放大后高清效果；
        
    3. 双击预览区域任意位置重置分割线，并进入放大细节检视模式。

4. **阿里云超分版本选择**

    1. 在设置页的阿里云服务配置中提供「生成式超分（默认）」与「标准版超分」两项，使用下拉选择控件，保持与其他服务字段同一层级；
    2. 选择生成式超分时，长宽比超过 `2:1` 且符合标准版尺寸限制的图片自动使用标准版，开始处理前和完成提示中说明实际使用版本；
    3. 选择标准版超分时，页面仅按标准版输入尺寸校验；不符合条件时在操作区域显示明确限制，不静默失败；
    4. 设置页服务列表与历史记录参数中应展示实际使用版本，自动切换时补充切换原因。

## 5. 状态管理与视觉反馈规范

所有交互组件状态需适配当前激活主题。

### 5.1 全局通知

- 全局成功与失败通知固定在右上角，位于自定义标题栏下方 `16px`，避免遮挡底部操作栏与任务队列；
- 通知从右侧滑入并自动消失；超分、生成等任务的详细状态仍在对应操作区域展示。

|   |   |   |   |
|---|---|---|---|
|组件 / 交互状态|深色主题样式|浅色主题样式|交互与动画规则|
|主操作按钮（常态）|背景`#6366F1` 文字`#FFFFFF`|背景`#4F46E5` 文字`#FFFFFF`|页面核心功能按钮基础样式|
|主操作按钮（悬浮 hover）|背景`#4F46E5`|背景`#4338CA`|过渡动画：`background-color 0.15s ease-in-out`|
|主操作按钮（禁用 / 加载中）|背景`#312E81` 鼠标样式：禁止|背景`#C7D2FE` 鼠标样式：禁止|按钮内显示环形加载动画|
|输入框（常态）|边框`#2E333F` 底色`#181A20`|边框`#E2E8F0` 底色`#FFFFFF`|普通输入框默认状态|
|输入框（聚焦）|边框`#6366F1` 外发光：`0 0 0 2px #4f46e533`|边框`#4F46E5` 外发光：`0 0 0 2px #4f46e533`|高亮边框，标识当前编辑输入框|
|输入框（校验错误）|边框`#EF4444` 辅助文字`#EF4444`|边框`#DC2626` 辅助文字`#DC2626`|参数校验失败时触发红色报错样式|

## 6. 桌面端专属交互（Tauri 框架原生能力）

充分调用 Tauri 系统集成接口，让网页 UI 拥有原生桌面软件体验：

1. **原生保存弹窗**：点击「下载」图标时，不使用浏览器下载逻辑；调用 Tauri 弹窗接口唤起系统「另存为」窗口，用户可自定义保存路径、修改文件名。
    
2. **拖拽上传区域**
    
    1. 超分页面支持本地文件拖拽导入；将桌面 / 文件管理器图片拖入窗口时，弹出蒙层提示：`拖放文件以开始超分处理`；
        
    2. 拖放目标区域以主强调色虚线边框高亮提示。
        
3. **右键上下文菜单** 右键任意预览图片，弹出适配当前主题的自定义菜单，包含：
    
    1. 图片另存为…
        
    2. 复制图片到剪贴板
        
    3. 在文件资源管理器 / 访达中打开（调用 Tauri 系统目录唤起指令）
        
4. **全局快捷键**
    
    1. `Cmd/Ctrl + G`：光标快速定位至提示词输入面板
        
    2. `Cmd/Ctrl + Enter`：直接执行「生成图像」主操作
        
    3. `Esc`：退出放大检视、关闭弹窗、终止正在运行的生成任务队列
